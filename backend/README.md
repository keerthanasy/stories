# AI Blog Backend

This is the backend for the AI-powered Blog Website, built with Node.js, Express, MongoDB, and OpenAI integration.

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- OpenAI (AI features)
- JWT (authentication)

## Getting Started

```bash
cd backend
npm install
npm run dev
```

The server will start on [http://localhost:5000](http://localhost:5000) by default.
